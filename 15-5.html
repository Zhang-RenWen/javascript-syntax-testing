15-5 Logical AND assignment(&&=)

Logical AND assignment &&= 左側的值若為 Truthy ,就會將 &&= 右側的值賦予 &&= 左側，並回傳賦值運算後的結果。 


let a = 1;
let b = 0;

a &&= 2;
console.log(a);
// Expected output: 2

b &&= 2;
console.log(b);
// Expected output: 0

語法


x &&= y

下列舉另外一種寫法表示，方便理解本章運算子的義涵。


x && (x = y);

x 為 Falsy 的話， &&= 左側的部份不會被賦值。 


const x = 0;
x &&= 2;
// Expected output: 0

配合物件修飾符 get/set 的範例


const x = {
  get value() {
    return 0;
  },
  set value(v) {
    console.log("Setter called");
  },
};

x.value &&= 2;

若 &&= 左側為 Falsy ，除了不會賦值外，事實上 &&= 右側的程式連執行都不會發生。 


const x = 0;
x &&= console.log("y evaluated");
// Logs nothing

更多示範案例加深理解：


let x = 0;
let y = 1;

x &&= 0; // 0
x &&= 1; // 0
y &&= 1; // 1
y &&= 0; // 0
