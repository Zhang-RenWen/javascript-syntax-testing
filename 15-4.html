15-4 Nullish Coalescing Assignment(??=)

判斷 ??=左邊變數為 空值(null or undefined) 對??=左邊的變數賦予??=右側的值。最後回傳賦值運算後的結果。

範例如下：


const a = { duration: 50 };

a.duration ??= 10;
console.log(a.duration);
// Expected output: 50

a.speed ??= 25;
console.log(a.speed);
// Expected output: 25

語法


x ??= y

下列舉另外一種寫法表示，方便理解本章運算子的義涵。


x ?? (x = y);

x 不為 空值時(null or undefined) 的話，??=左側的部份不會被賦值。


const x = 1;
x ??= 2;
// Expected output: 1

配合物件修飾符 get/set 的範例


const x = {
  get value() {
    return 1;
  },
  set value(v) {
    console.log("Setter called");
  },
};

x.value ??= 2;
// Expected output: 1

若 ??= 左側為 空值時(null or undefined)，除了不會賦值外，事實上??=右側的程式連執行都不會發生。


const x = 1;
x ??= console.log("y evaluated");
// Logs nothing

有用的預設值設定應用示範


function config(options) {
  options.duration ??= 100;
  options.speed ??= 25;
  return options;
}

config({ duration: 125 }); // { duration: 125, speed: 25 }
config({}); // { duration: 100, speed: 25 }
