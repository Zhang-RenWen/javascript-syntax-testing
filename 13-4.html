13-4 String.prototype.trimEnd()

trimEnd() 方法會刪除字串末尾的空白字符。trimRight() 是這個方法的別名。


const greeting = '   Hello world!   ';

console.log(greeting);
// Expected output: "   Hello world!   ";

console.log(greeting.trimEnd());
// Expected output: "   Hello world!";

別名


trimEnd()

trimRight()

為了與 String.prototype.padEnd 等函數保持一致，標準方法名稱為 trimEnd。但是，出於 Web 兼容性原因，trimRight 仍然是 trimEnd 的別名。在某些引擎中，這意味著： 


String.prototype.trimRight.name === "trimEnd";

範例


let str = "   foo  ";

console.log(str.length); // 8

str = str.trimEnd();
console.log(str.length); // 6
console.log(str); // '   foo'
